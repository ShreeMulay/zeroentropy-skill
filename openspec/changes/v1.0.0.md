# Change: v1.0.0 Initial Release

## Status
completed

## Description
Initial release of the ZeroEntropy Agent Skill providing comprehensive coverage of ZeroEntropy's retrieval infrastructure for AI agents.

## Changes

### Added
- SKILL.md with frontmatter, routing, and comprehensive pitfalls documentation
- 5 recipe guides: embedding, indexing, searching, reranking, RAG pipeline
- Python examples (01-05) with utils (retry, chunk)
- TypeScript examples (01-05) with utils (retry, chunk)
- JSON schemas: metadata-filter.json, index-config.json
- Test suite: skill lint validation, API contract integration tests
- CI/CD: GitHub Actions for lint, validate, typecheck, integration
- Release workflow: version validation and GitHub releases
- skill.json for agent discovery
- README.md with installation instructions
- CHANGELOG.md
- LICENSE (MIT)
- OpenSpec documentation

### Recipes
1. **01-embedding.md** — zembed-1 usage, dimensions, encoding formats, latency modes
2. **02-indexing.md** — Collection creation, document upload (text, binary, pages), metadata rules
3. **03-searching.md** — Query granularity (documents, pages, snippets), metadata filtering
4. **04-reranking.md** — zerank-2 usage, score semantics, two-stage pipeline
5. **05-rag-pipeline.md** — End-to-end RAG with polling, reranking, context building

### Pitfalls Documented
- `list:` metadata prefix (silent failure)
- Embedding dimension lock (silent corruption)
- Rerank score semantics (relative, not absolute)
- Async indexing (eventual consistency)
- ConflictError handling (409)
- Rate limits (429, backoff)
- Latency modes (low vs high)
- Token truncation
